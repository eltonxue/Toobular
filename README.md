# Toobular
Interactive website that utilizes the public YouTube API to simulate video hosting. Built primarily using React.

# To run:
Run this command:
```
npm start
```
Then, visit http://localhost:8080

# Technologies Used

Front-end:
- HTML/CSS
- React

Back-end:
- Node.js

Packages:
- Webpack
- Babel
- Lodash

APIs:
- YouTube API
